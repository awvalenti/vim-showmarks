# SHOW MARKS

show marks on signcolumn for vim or nvim

fast, mini, simple

## INSTALL

`plug 'yaocccc/vim-showmarks'`

## CONFIG

```viml
  hi Marks ctermfg=80
  let s:mark_ns_id = [number]  " default 9898
  let s:mark_priority = [number] " default 999
  let s:enabled_marks = "[string]" default '[a-zA-Z]'
```
